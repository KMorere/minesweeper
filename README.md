# Démineur :
## Description :
Réalisation d'un démineur en java dans la console.
Il suffit de saisir les coordonnées souhaitées dans la console,  
(Ex:1 1 pour la cellule en haut à droite)  
Si vous touchez une cellule vide alors celles avoisinantes seront révélées.  
La partie se termine une fois toutes la grille découverte ou une bombe est touchée.
